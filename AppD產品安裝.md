# AppD產品安裝

## Agent安裝 
- Java 1.6~11
- NgwinX 有支援
- 一般Linux的Machine Agent都含有JRE，若為AIX需自行安裝JRE1.8
- Docker Container環境不適用Machine Agent（其實Docker自行用OSS監控即可）
- 安裝Agent需要key，Key需要從管理網頁的Account取得
- 各單位的APM窗口有Account權限
